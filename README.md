# Naman Trisoliya - Portfolio

A clean, modern, and responsive portfolio website built with **Angular 19** and **Angular Material**.

## 🚀 Live Demo
[namant-portfolio.vercel.app](https://namant-portfolio.vercel.app/)

## ✨ Features
- **Modern UI/UX**: Clean design with smooth scroll animations.
- **Responsive Design**: Optimized for desktops, tablets, and mobile devices.
- **Dark Mode**: Toggle between light and dark themes.
- **Dynamic Typing Effect**: Interactive hero section with a typing animation.
- **Project Showcase**: Filterable portfolio projects (WIP).
- **Interactive Resume**: Professional experience and education timeline.

## 🛠️ Tech Stack
- **Framework**: Angular 19+
- **Styling**: SCSS, Angular Material
- **Icons**: FontAwesome, Material Icons
- **Deployment**: Vercel

## 📂 Project Structure
- `src/app/core`: Shared components like Navigation and Footer.
- `src/app/features`: Main content sections (About, Resume, Skills, Portfolio, Contact).
- `src/app/features/main`: Container component for the single-page layout.

## 🛠️ Installation & Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/NamanT98/Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm start
   ```
   Open `http://localhost:4200/` in your browser.

## 🏗️ Build
To build the project for production:
```bash
npm run build
```
The build artifacts will be stored in the `dist/` directory.
